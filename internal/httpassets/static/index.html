Waypoint is a tool built by HashiCorp for building, deploying and releasing applications.

Applications deployed with Waypoint receive a public waypoint.run URL like this one, for viewing and sharing deployed applications.

Waypoint allows developers to define their application build, deploy, and release lifecycle as code, reducing the time to deliver deployments through a consistent and repeatable workflow.
